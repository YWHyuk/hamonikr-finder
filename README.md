### hamonikr-finder
hamonikr-finder란.<br>
하모니카 파인더는 데스크톱과 클라우드 저장소에서 1400개 콘텐츠 형식(hwp 파일 포함)에 대하여 원문을 검색할 수 있는 프로그램입니다.
각각의 모듈은 도커 컨테이너 형태로 독립적인 서비스를 제공하도록 구성하여 추후 재사용성을 높일 수 있습니다.
